# v0.5.0 — Stable & Easy Mode

这是一次稳定易用性升级，核心目标是让 Novel Pipeline Write Engine **跑得通、看得懂、改得动**。

## 核心亮点

| 模块 | 说明 |
|------|------|
| 统一入口 `novel.py` | init / demo / status / pre / post / review / report / export / guards / check |
| 健康检查 | `python novel.py status` 一键诊断环境 |
| 写前任务卡 | 自动从 SQLite 提取上下文，生成承接/推进/禁止 + Voice/Meme 提醒 |
| Reader Pull Guard | 追读力门禁（钩子/兑现/悬念/爽点/代价） |

| Voice/Meme Pack 增强 | YAML 格式，41 个语言包，加载器 + 校验器 |
| 5 种题材模板 | 修仙/都市异能/规则怪谈/悬疑/科幻 |
| Windows 一键体验 | install.bat → run_demo.bat → run_report.bat |

## 新增功能

### 统一命令入口 novel.py

v0.4.x 的 init / pre / post / review / report 散落在不同脚本中，需要手动传 config 和 slug。v0.5.0 统一为一个入口：

```bash
python novel.py status              # 环境健康检查
python novel.py demo                # 跑通 Demo
python novel.py pre 1               # 第 1 章写前任务卡
python novel.py post 1              # 入库 + 门禁
python novel.py review 1            # 审查
python novel.py report              # HTML 报告
python novel.py export              # 导出小说
python novel.py guards              # 列出已注册门禁
python novel.py check chapter.txt   # 单文件门禁检查
```

### status 健康检查

`python novel.py status` 检查项：

- Python 版本（≥ 3.10）
- config.json 存在性
- 核心 guard 模块存在性（reader_pull_guard, voice_pack_guard, meme_pack_guard）
- voice_packs/ 目录完整性
- templates/ 目录完整性
- SQLite 数据库
- Python 依赖安装状态

### 写前任务卡 Task Card

`python novel.py pre <N>` 自动从 SQLite 提取：

- **承接项**：上章结尾状态、未解决钩子、进行中任务
- **推进项**：本章标题、章节目标、预期方向
- **禁止项**：不能出现的内容和状态
- **Voice 提醒**：角色应使用的声纹包
- **Meme 提醒**：可用的梗包及浓度限制

输出：`outputs/task_cards/chapter_<N>_task_card.md`

### Reader Pull Guard 追读力门禁

v0.5.0 核心新增门禁，专门检查网文追读力 5 维度：

| 维度 | 检查内容 | 输出 |
|------|----------|------|
| 钩子设置 | 章末是否设置新钩子 | PASS / WARNING / FAIL + 行号 |
| 钩子兑现 | 上章钩子是否得到回应 | PASS / WARNING / FAIL + 行号 |
| 悬念推进 | 主线悬念有无实质推进 | PASS / WARNING / FAIL + 行号 |
| 爽点落地 | 打脸/升级/反转/收获 | PASS / WARNING / FAIL + 行号 |
| 代价展示 | 行动带来的可见代价 | PASS / WARNING / FAIL + 行号 |

### 章节风险评分 risk_score.py

8 维度评分（词数 10%、场景数 15%、钩子 20%、AI 腔 15%、水文 15%、对白密度 10%、破折号密度 5%、梗浓度 10%），综合评分 A~F。

### HTML 只读报告

- 纯静态 HTML，无 JavaScript CDN 依赖
- 双击 `run_report.bat` 自动打开 `reports/index.html`
- 完全离线工作
- 可分享给审稿人

### Voice Pack 增强（41 个语言包）

YAML 格式替代 JSON，核心与小说角色解耦：

- `voice_packs/base/` — 9 个通用角色声纹包
- `voice_packs/registers/` — 8 个语体包
- `voice_packs/dialects/` — 6 个方言包
- `voice_packs/memes/` — 2 个梗包
- `voice_packs/english/` — 2 个英语包
- 旁白/禁用词

配套加载器：`src/voice/voice_pack_loader.py` + `src/voice/voice_pack_validator.py`

### Meme Pack 增强

- **梗浓度控制**：可配置最大梗密度
- **角色绑定**：梗包绑定到特定角色类型
- **场景绑定**：按场景类型控制梗使用
- **冷静期**：连续梗后强制间隔
- **banned_memes**：高频网络热梗禁用库

配套加载器：`src/meme/meme_pack_loader.py` + `src/meme/meme_pack_validator.py`

### 对白密度门禁
- >10% 行数为对白 → WARNING
- >20% 行数为对白 → SEVERE

### 破折号密度门禁
- >5/千字 → WARNING
- >12/千字 → SEVERE

### 无引号对白检测
- fallback 模式，检测无引号标记的对白行
- 适应部分网文不使用引号的写作风格

### 代码结构模块化
- `src/` 目录按功能拆分：cli / guards / task_card / voice / meme / report
- 替代 v0.4.x 的扁平 scripts/ 目录
