Novel Pipeline WebUI v0.6.6 Browser-First 扫描修正版

这版的目标：普通用户打开就能用，不再因为“后端未启动 / 引擎未接”卡在第一步。

推荐使用方式：
1. 解压整个 zip，不要只拖出 webui 文件夹。
2. 双击：一键启动_WebUI.bat
3. 它会自动打开浏览器。
4. 如果 5173 已被旧版本占用，它会自动改用 5175，避免你又打开旧版页面。

普通用户流程：
1. 新建项目默认选择“浏览器项目”。
2. 题材默认“修仙 / 玄幻”，风格默认“网络小说”。
3. 填小说名、作者名、大纲后就能创建。
4. 文库 / 独立审稿里点“选择本地文件夹”，浏览器会读取 txt/md 章节。
5. 不启动后端也能扫描、预览、导出、轻量审稿。

高级用户流程：
1. 需要真正写入 D 盘目录时，切换到“磁盘项目”。
2. 填完整路径，例如 D:\Novels\我的小说。
3. 确保后端窗口没有报错，顶部显示后端 v0.6.6。

重要变化：
- 不再连接旧版 8787 后端，避免误显示“v0.6.0-webui-preview”。
- 顶部如果显示“旧后端”，说明你还有旧黑窗口没关，关闭旧窗口后重启。
- “引擎未接”不再阻塞普通用户，浏览器模式仍然可用。
- 新建项目默认不弹后端目录浏览器，不再出现一打开就“后端没有连上”的红框。
