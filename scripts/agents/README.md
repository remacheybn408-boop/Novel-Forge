# Multi-Agent System (Planned)

Multi-agent review is not enabled in v0.5.0.
v0.5.0 focuses on stable single-pipeline execution.
The agents directory is only a placeholder for future versions.

Planned for v0.5.1 or v0.6.0:
- Agent orchestration for distributed guard execution
- Multi-perspective review (character consistency agent, plot coherence agent, etc.)
- Agent conflict resolution

For now, all guard execution runs through the single-pipeline `guard_registry`.
