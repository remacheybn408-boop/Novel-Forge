## 关于 v0.8.0

Novel Forge v0.8.0 是在 v0.7.x 开源版（已在 main 分支归档）基础上全新研发的商业版本。核心引擎全面重构，从 CLI 工具进化为完整的桌面写作平台。

### 核心升级

**全新桌面客户端**
- 基于 PySide6 的原生桌面应用，告别终端操作
- 暗色玻璃质感主题，沉浸式写作界面
- 实时状态面板：字数额度、Guard 检测结果、章节连贯性一目了然
- 左侧导航 + 多页面布局：仪表盘 / 写作 / 角色 / 大纲 / 档案 / 质量报告 / 设置

**写作体验提升**
- 自动保存机制，停止输入即保存，覆盖前轻量备份
- 润色 / 扩写 / 缩写 / 重写入口，结果预览后确认，不自动覆盖正文
- 右侧实时上下文面板：展示当前章节命中的角色、地点、关联设定
- 兼容纯 CLI 模式，桌面端和命令行并行可用

**质量体系升级**
- Guard 门禁系统扩展至 30+ 项检测
- Agent 陪审团全面优化，评审维度更细，误报率更低
- 一键质量报告：聚合 Guard 评分、Agent 审稿分、故事链健康度、角色弧线完整度
- 发布前完整验收流程：稳定性检查 + 交叉平台验证

**故事管理增强**
- 多小说独立 slot 数据库，内容互不干扰
- 大纲版本管理，支持版本对比与回退
- 角色卡五维系统：声纹 / 性格 / 做事风格 / 叙事层 / 精神状态
- Story Contract 章节合约链，长篇不丢伏笔

**部署与兼容**
- 支持 Windows / macOS / Linux 三平台
- 提供一键启动器，零配置上手
- 兼容 uv 与 pip 两种部署方式
