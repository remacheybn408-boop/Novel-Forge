# v0.4.0 — Human-Grade Revision Suite

v0.4.0 增加拟人审稿质量套件。通过具体锚点、场景因果、对白自然度、句式变化、审稿痕迹、合规自查和最终投稿报告，提升小说稿件的自然度、原创感、连续性和投稿前可检查性。

本项目不提供 AI 率、人类率或平台过检率。

## 新增 Guards

| Guard | 功能 | 级别 |
|-------|------|------|
| `editor_revision_guard` | 审稿痕迹检查：检测过度解释/初稿感 | WARNING only |
| `concrete_anchor_guard` | 具体锚点密度：物件/动作/场景锚点 | WARNING only |
| `scene_causality_guard` | 场景因果链 CARCRH：原因/行动/阻力/代价/结果 | WARNING only |
| `dialogue_naturalness_guard` | 对白自然度：打断/未完成句/动作节拍 | WARNING only |
| `style_variation_guard` | 句式变化：开头重复/抽象词/句长变化度 | WARNING only |
| `compliance_selfcheck_guard` | 投稿合规自查：高风险可 BLOCK 入库 | 可 BLOCK |
| `final_submission_report` | 最终投稿报告：汇总门禁 + 投稿建议 | — |

## Revision Loop 自动改稿闭环

把最终投稿报告中的高优先级问题转换成修改任务，在 controlled 模式下生成可审阅的 revised draft 和 diff report。

- 默认不会覆盖原文
- 默认不会自动入库
- 默认最多改 2 轮
- 默认只修 Top 5 高置信度问题
- 如果改稿导致指标变差，系统标记 REVISION_REJECTED

Revision Loop 不是自动洗稿工具。它的目标是辅助作者审稿和改稿。
